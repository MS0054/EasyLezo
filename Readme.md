# MyComposeApp

A modern Android application built with Kotlin and Jetpack Compose.

---

## 📱 Description
This project is an Android application developed using **Kotlin** and **Jetpack Compose**.  
The goal of this project is to practice modern Android development principles and architecture.

## 🛠 Tech Stack
- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Coroutines
- Flow

---

## 🧱 Architecture
The project follows **MVVM (Model–View–ViewModel)** architecture principles:

- UI layer built with Jetpack Compose
- ViewModel for state management
- Clean separation of concerns

---

## ⚙️ Requirements
- Android Studio (Hedgehog or newer recommended)
- Minimum SDK: 24
- Gradle 8+

---

## ▶️ How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Click **Run**
